# Mycolibrium

A **work‑in‑progress** web app that predicts when (and where) wild mushrooms will fruit in New England by combining recent weather data with community observations.

---

## What it does (MVP)

* Pulls **current & recent weather** from [Open‑Meteo](https://open-meteo.com)  
* Shows a simple “chance of fruiting” indicator on a map
